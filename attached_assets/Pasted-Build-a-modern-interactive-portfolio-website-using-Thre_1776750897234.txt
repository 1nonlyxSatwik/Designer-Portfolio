Build a modern interactive portfolio website using Three.js (WebGL) with smooth animations and a premium dark UI.

The goal is to create a visually impressive portfolio that feels like a high-end designer website, while keeping performance stable and code simple.

TECH STACK:

* HTML, CSS, JavaScript
* Three.js (for WebGL background)
* GSAP + ScrollTrigger (for animations)
* No frameworks (no React)

---

GLOBAL DESIGN SYSTEM:

* Background: #0B0B0F

* Secondary background: #12121A

* Accent gradient: #FF3D77 → #6C5CE7

* Text primary: #FFFFFF

* Text secondary: #A0A0B2

* Border: rgba(255,255,255,0.08)

* Font: Inter / system sans-serif

* Border radius: 16px–24px

* Smooth spacing (section padding: 100px+)

---

WEBGL (THREE.JS) BACKGROUND:

* Fullscreen canvas fixed in background
* Create a simple floating 3D scene:

  * 3–5 abstract spheres or blobs
  * Use MeshStandardMaterial (matte + slight metalness)
  * Soft lighting (ambient + point light)
* Objects should:

  * Slowly rotate
  * Float gently (sin wave motion)
* Mouse movement:

  * Slight parallax effect (camera shifts slightly)
* Keep polygon count LOW for performance
