## ✨ Overview

**Fluxfolio** is not a conventional portfolio.

It was built as an interactive digital experience where the interface itself tells the story.

Rather than displaying work through static case studies, this project explores:

- Product Design  
- Creative Development  
- Interactive Storytelling  
- Motion Systems  
- Experimental Interfaces

# 📂 Installation

Clone repository:

```bash
git clone https://github.com/yourusername/fluxfolio.git
```

Move inside:

```bash
cd fluxfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
